# Heart Disease Prediction

A Python web app that predicts the risk of heart disease using machine learning.

## Project Overview

This project uses a dataset of heart health parameters to train a machine learning model. The app allows users to input their health data and predicts the likelihood of heart disease.

## Project Structure

- `app.py` – Flask web application for user interface  
- `train_model.py` – Script to train and save the ML model  
- `model.pkl` – Trained machine learning model  
- `heart.csv` – Dataset used for training  
- `requirements.txt` – Python dependencies  
- `templates/` – HTML templates for the web interface
